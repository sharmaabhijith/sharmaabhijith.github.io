This project was a part of Mini Project that I did during my junior year. This project had opened up the door of interest in me for control systems and electronics. 
We made a prototype to demonstrate magnetic leviataion of a small mass using PID control and ON/OFF control. This Real Time data was uploaded continuously on 
Think Speak platform using IOT module ESP8266. We also designed a application specific board for this with Atmega 32 controller. 
<br />
This work wouldn't been possible without the collaborative efforts of Abhishek Tambe and Shubham Ganu.
